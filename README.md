# AI経費処理システム

> **開発コード名：AI経費処理システム**

AIを活用して経費処理業務を支援するシステムです。

本システムは、領収書・レシート・クレジットカード利用明細などの経費関連情報を解析し、共通経費データとして管理するとともに、AIによる継続的な経費チェックを行い、各種会計ソフトと連携することを目的としています。

---

# プロジェクトの目的

本システムは、経費処理業務の効率化だけでなく、AIによる経費チェックを通じて経費処理品質の向上とコスト削減を支援することを目的としています。

目標

* 領収書入力の自動化
* 経費データの一元管理
* 重複登録・登録漏れ防止
* クレジットカード利用明細との照合
* サブスクリプション契約の継続監視
* 利用料金の変動検知
* 不要な契約や異常支出の検知
* 会計ソフトとの連携

---

# 主な機能

### 経費入力

* 領収書・レシートのアップロード
* 領収書情報の自動読取
* 経費情報の自動入力
* 利用者による確認・修正

### 経費管理

* 共通経費データ管理
* クレジットカード利用明細取込
* 領収書との照合
* 重複登録チェック
* 登録漏れチェック

### AIチェック

* 勘定科目提案
* 支払方法判定
* 経費内容チェック
* 継続課金検知
* 利用料金変動検知
* 異常支出検知

### 会計ソフト連携

* 会計ソフト向けデータ生成
* 会計ソフト別フォーマット変換
* 会計事務所提出データ作成

# システム構成

```text
利用者
      │
      ▼
AI解析
      │
      ▼
共通経費データ
      │
      ├── 経費管理
      ├── AIチェック
      └── 会計ソフト連携
              ├── TKC FX2
              ├── 弥生会計
              ├── freee（予定）
              ├── マネーフォワード（予定）
              └── その他
```

# 開発方針

本システムは、特定の会計ソフトや実装技術に依存しない設計を採用している。

共通経費データを中心に設計し、会計ソフト固有の仕様は Exporter が吸収する。

また、入力支援だけでなく、AIによる継続的な経費チェックと改善提案を通じて、経費処理品質の向上を目指す。
