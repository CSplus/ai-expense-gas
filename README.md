# AI経費領収書管理システム

## 概要

Google Apps Script と Gemini API を利用したAI経費管理システムです。

## 主な機能

- 領収書OCR
- Gemini解析
- 経費台帳登録
- 重複チェック
- カード明細取込
- 月次CSV出力
- 会計事務所メール送信
