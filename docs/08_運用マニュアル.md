# 運用マニュアル

## 1. この文書について

本書は、現行GWS版の運用に必要な最低限の項目を整理する。

詳細な画面操作、トリガー設定、障害対応手順は今後整備する。

## 2. 現行GWS版の主な運用対象

* Google Apps Script
* Google Sheets
* Google Drive
* Gemini API
* Gmail 送信

## 3. 事前設定

現行GWS版では、以下の設定が必要となる。

* `GEMINI_API_KEY` を Apps Script のスクリプトプロパティへ設定する。
* `システム設定` シートに各種フォルダID、通知先メール、Geminiモデルなどを設定する。
* `経費台帳`、`仕分けルール`、`勘定科目マスタ`、`税率マスタ` を準備する。

## 4. 主な運用作業

* 領収書画像のアップロード
* クレジットカードCSVの未処理フォルダへの配置
* カード明細取込処理の実行
* 重複検出処理の実行
* 月次集計の作成
* 月次経費報告メールの送信
