# 弥生会計連携仕様

## 1. この文書について

本書は、AI経費処理システムから弥生会計へ連携するための仕様を整理するための文書である。

現時点では本格実装は未実装であり、詳細仕様は今後対応とする。

## 2. 基本方針

* 共通経費データをもとに弥生会計向けデータを生成する。
* 弥生会計固有のコード、税区分、出力形式は Exporter 側で吸収する。
* 共通経費データ自体を弥生会計専用形式にしない。

## 3. 現行GWS版での扱い

現行GWS版では、既存動作維持のため経費台帳に `ACCOUNT_CODE` を保持している。

ただし、将来の複数会計ソフト対応では、会計ソフト固有のコード変換は Exporter 側へ移管する。

## 4. 未実装・今後対応

以下は未実装または今後対応とする。

* 弥生会計インポート形式の確定
* 勘定科目コード変換表
* 税区分コード変換表
* 補助科目の扱い
* 出力ファイルの検証手順
